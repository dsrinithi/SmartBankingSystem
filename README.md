# 🏦 Smart Banking System

A Java Desktop Banking Application developed using JavaFX. This project provides a simple and secure banking system where users can perform various banking operations through an interactive graphical interface.

---

## 📌 Features

- 🔐 User Login
- 💰 Deposit Money
- 💸 Withdraw Money
- 📊 Balance Enquiry
- 📝 Transaction History
- 💾 File Handling for Data Storage
- 🎨 User-friendly JavaFX Interface

---

## 🛠 Technologies Used

- Java
- JavaFX
- FXML
- CSS
- File Handling
- Eclipse IDE

## 🚀 How to Run

1. Clone this repository

```
git clone https://github.com/YourUsername/SmartBankingSystem.git
```

2. Open the project in Eclipse IDE.

3. Configure JavaFX libraries.

4. Run Main.java
